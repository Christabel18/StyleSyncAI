# Product Images

Drop product images here for the OutFitted shop pages.

| File | Used by | Description |
|------|---------|-------------|
| `yellow-hoodie.jpg` | `/shop` | Vibrant Yellow Hoodie — Nova's pick for streetwear |

## Adding new images
1. Place image in this folder (JPG or PNG, ideally square or 4:5 ratio)
2. Reference in Next.js as `/images/filename.jpg` (no `/public` prefix)
3. Use the `<Image>` component from `next/image` with `unoptimized` for external sources
